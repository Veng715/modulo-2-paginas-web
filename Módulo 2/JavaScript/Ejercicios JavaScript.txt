Variables y Tipos de Datos

mensaje de saludo que incluya su nombre.


Pide al usuario dos números usando prompt(). Conviértelos a tipo numérico y muestra la 
suma en la consola o en un alert.

Pide al usuario su edad usando prompt(). Guarda este valor en una variable.

Declara variables para guardar el nombre, la especie y la edad de una mascota. Asígnales 
valores y luego muéstralos concatenados en un mensaje.

imprime el resultado para ver cómo JavaScript maneja la conversión.



Condicionales

Usando una variable de edad (puedes pedirla con prompt()), verifica si el usuario es mayor o 
igual a 18 años. Muestra un mensaje indicando si puede conducir o no.

Pide un número al usuario con prompt(). Usa if-else if-else para determinar si el número es 
positivo, negativo o cero, y muestra el resultado.

Pide al usuario que ingrese una contraseña con prompt(). Si la contraseña es "secreto123", 
muestra "Acceso concedido". De lo contrario, muestra "Acceso denegado".

Pide al usuario un número del 1 al 7 con prompt(). Usa una estructura switch para mostrar el 
día de la semana correspondiente (1 para Lunes, 2 para Martes, etc.).

Pide una calificación numérica (0-10) con prompt(). Si la calificación es 7 o más, muestra 
"Aprobado". Si es menor, muestra "Reprobado".
